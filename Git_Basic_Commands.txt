Git Commands
************
git add .
git commit -am "commit message"
git push origin master
git clone "clone URL"

…or create a new repository on the command line
git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/anandarajRepo/TradingviewAlgorithum.git
git push -u origin master

…or push an existing repository from the command line
git remote add origin https://github.com/anandarajRepo/TradingviewAlgorithum.git
git push -u origin master

Git clone repository
--------------------
git clone <remote_repo>
git clone -b <branch> <remote_repo> --> Clone form branch

Basic setup
-----------
gir config --list --show-origin
git config --global user.name "User Name"
git config --global user.email "email"

Colours
-------
git config --global color.ui true

Initialising git
----------------
git init

Adding and committing
---------------------
git add . --> Adds all file
git add *.txt --> Adds all .txt extension file
git add <file> --> Adds a specific file
git commit -m 'first commit'

Discard changes in working directory
------------------------------------
git discard -- <file>

Push
----
git push origin master

change remote URL
-----------------
git remote set-url origin https://github.com/GeorgeSeif/Semantic-Segmentation-Suite.git

Remote backup
-------------
### General example
git remote add origin https://github.com/user/repo.git

### An example with a repository of mine
git remote add origin https://github.com/GeorgeSeif/Semantic-Segmentation-Suite.git

Pull Changes made in repository
-------------------------------
git pull
git pull --verbose

## Gives verbose output during a pull which displays the content being downloaded and the merge details.

Branch and Fork
---------------
Branch will create a clone in same repository. While, Fork will clone a file in someother acct's repository
